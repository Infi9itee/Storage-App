# Storage-App

Simple initial concept of a storage app where users can store text or image files.

**Instructions:**
1. To run this simply download all the files in a folder.
2. All the files needs to be in the same folder to run all the pages.
3. Click on h1 html, a browser tab will open and from there user can browse the app.
4. For now there's no set username or password, so users can enter anything to enter the app.
